### 프로젝트 배경 🧑‍🦯

시각 장애인을 위한 의약품 점자 표기 부재로 인한 의약품 오류복용 사고가 빈번하게 발생하고 있다. 이러한 문제를 해결할 수 있는 시각장애인 전용 서비스를 개발함으로 사용자의 편의성을 확보하고자 한다.

<br>

### 프로젝트 내용 💊

네이버 AI 클라우드와 외부 API 연동을 통해 사용자의 증상에 적합한 의약품 정보 제공하기

**서비스 로드맵**

<img width="90%" alt="이약점프로세스" src="https://user-images.githubusercontent.com/87962966/145804552-76f38c3b-6221-42a9-bd4e-1e1173706ac3.png">

<br>

총 4개의 기능별 서비스 모듈을 구성하고 이에 맞게 서비스 프로세스 구축

1. 음성 인식 모듈 (증상 입력)
2. 사진 판독 모듈 (약품 사진 입력)
3. 약품데이터 정보 제공
4. 음성 안내 모듈
