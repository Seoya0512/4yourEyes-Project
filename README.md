## 네이버 클라우드 서비스를 활용한 AI서비스 개발 PJT



#### ✏️ 프로젝트 제목 

시각장애인을 위한 의약품 정보 제공 서비스 개발


#### 🗓 수행 기간 

'21. 10. 18 ~ '21.10. 28



#### 💻 수행 도구 

- 분석 기술 패키지 : Pandas, Numpy, KoNLPy

- 시각화 기술 패키지 : Matplotlib, Seaborn

- 수집기술 패키지 : BeaurifulSoup

- AI Service : 네이버 클라우드 (OCR, Speech, Voice)

  

#### 📦 사용 데이터
- 의약품 안전나라 공공 API
